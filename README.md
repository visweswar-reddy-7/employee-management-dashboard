# Employee Management Dashboard

## Overview
A modern, responsive HR dashboard built with React and Vite. Features include authentication, full CRUD operations for employees, combined filtering, and a print-ready list.

## Tech Stack
- **React 18** (Vite)
- **Tailwind CSS** (Styling)
- **Lucide React** (Icons)
- **Vitest** (Testing)
- **Local Storage** (Data Persistence)

## Running Locally
1. `npm install`
2. `npm run dev`
3. Log in with: `admin@example.com` / `password123`

## Assumptions
- Browser Local Storage is used for data persistence to fulfill the mock API requirement.
- Full Name and DOB are mandatory for record creation.
